# Authorization Code Flow for Web Apps - OAuth0

1. Crie o web app

    ![alt text](<Screenshot from 2025-07-06 15-36-55.png>)

2. Adicione a URL de redirecionamento de login

    ![alt text](<Screenshot from 2025-07-06 15-37-31.png>)

3. Gere o código `PKCE` (String de 43 a 128 caracteres aleatórios)
    - 4556c5beadf2c8dd029a07ebde9dbefc70a9a6b9fdc2428bb2f81c55

4. Gere o `Challenge Code` (que é o hash SHA256 codificado em Base64-URL da string aleatória gerada):
    - rAIiN7ymoqur4JvJz5Xnj07PjO81eHaOQVuMDyhgHkE

5. Crie a URL:
    ```
    https://dev-lyx6hvf82sns21u1.us.auth0.com/authorize?response_type=code&client_id=xDCCjHAVFu7Zm97leM0R8psMLm7qQBUg&
    redirect_uri=https://webapp-examplee.com/redirect&
    state=4556c5beadf2c8dd029a07ebde9dbefc70a9a6b9fdc2428bb2f81c55&code_challenge=rAIiN7ymoqur4JvJz5Xnj07PjO81eHaOQVuMDyhgHkE&code_challenge_method=S256
    ```

6. Faça o login:

    ![alt text](<Screenshot from 2025-07-06 15-42-15.png>)

7. Após receber o Authorization Code:

    ![alt text](<Screenshot from 2025-07-06 15-43-12.png>)

    ```http
    curl -X POST https://dev-lyx6hvf82sns21u1.us.auth0.com/oauth/token \
    -d grant_type=authorization_code \
    -d redirect_uri=https://example-app.com/redirect \
    -d client_id={YOUR_CLIENT_ID} \
    -d client_secret={YOUR_CLIENT_SECRET} \
    -d code_verifier={YOUR_CODE_VERIFIER} \
    -d code={YOUR_AUTHORIZATION_CODE}
    ```
